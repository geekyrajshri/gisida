Map visualizations from
MapSpec using Mapbox GL JS

Gisida is a javascript library that makes it easy to create beautiful map visualizations from data represented in a standardized way. Gisida extends Mapbox Style Specification to use simplify the process of creating popular map visualizations like choropleth with autobreaks with support for csv data joins, timeseries support, etc.
